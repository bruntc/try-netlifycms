# Try NetlifyCMS

## tl;dr

Build & serve

    npm install
    npm run develop

And access site with http://localhost:8000/

Access CMS at http://localhost:8000/admin/

## Rebuild

After content change has been published, the local app can rebuilt with

    git pull
    npm run develop
